Test the SD Card DiskIO functionalities Low level disk I/O module function checker.

Note
WARNING: The data on the target drive will be lost! int rc; DWORD buff[EF_MAX_SS]; Working buffer (4 sector in size)

Check function/compatibility of the physical drive #0

rc = s32TestSdCardDiskIO(0, 3, buff, sizeof buff);

if (rc) { printf( "Sorry the function/compatibility test failed. (rc=%d)\nFatFs will not work with this disk driver.\n", rc ); } else { printf( "Congratulations! The disk driver works well.\n" ); }
