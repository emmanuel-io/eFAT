◆ EF_LFN_BUF

#define EF_LFN_BUF   ( 255 )

This set of options defines size of file name members in the ef_file_info_st structure which is used to read out directory items. These values should be sufficient for the file names to read. The maximum possible length of the read file name depends on character encoding. When LFN is not enabled, these options have no effect.
